Objective
	This repo is the central repository for all the casestudy1 artifact.

Content
	CaseStudy
	This folder contains the Case Study01.docx which is the requirements specification that was receieved to build the casestudy

	Rmd
	This folder contains the knitted rmarkdown file along with it's html version.All the R code for the case study resides in Rmarkdown file

	Exploratory Data Analysis.pdf
	This is the final casestudy presentation copy.
	
	Codebook
	This is the technical documentation for the casestudy
